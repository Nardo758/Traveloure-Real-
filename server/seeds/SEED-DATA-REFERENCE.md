## 🔄 Resetting Data

To completely reset and reseed:

```bash
# CAUTION: This deletes all data!

# Option 1: Reset database
npm run db:reset
npm run db:migrate
npm run seed:beta

# Option 2: Manual delete (using Drizzle Studio)
npm run db:studio
# Delete data manually from tables
```
